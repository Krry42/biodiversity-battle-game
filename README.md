## üß© About biodiversity-battle-game

Biodiversity Battle is an online interactive board game created for the Science Education PhD Program (PPGEduC) at the University of Brasilia (UnB). It aims to teach players about biodiversity through engaging gameplay. Developed by Lucas Ferreira (UnB/PPGEduC), the game blends biology concepts and strategic moves to help users learn about living things and ecosystems.

This game serves as an educational tool for students, teachers, and anyone curious about biology. It makes the study of biodiversity interesting and approachable by offering a virtual board game experience.
